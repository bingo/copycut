# CopyCut — 项目概览

## 产品定位

CopyCut 是一款面向内容创作者的 Web 端视频/图文编辑工具，以剪映 Web 版为参照原型，针对**小红书平台内容生产场景**深度裁剪与优化。

Alpha 版本目标用户：在小红书上发布生活方式、美妆、旅行、美食、知识分享类内容的普通创作者，无需专业剪辑背景即可快速出片。

当前文档对应项目 Step 1：完成剪映 Web 能力分析和小红书场景裁剪。Alpha 版本在 Step 2 中先实现 Presentation 层 Mock，剪辑、AI、真实导出等后台能力在 Step 3 接入。

---

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 人群 | 18–35 岁，小红书活跃内容创作者 |
| 技能 | 非专业剪辑，熟悉手机端短视频制作 |
| 设备 | MacBook / Windows PC，Chrome/Safari 浏览器 |
| 内容类型 | vlog、穿搭、美食探店、旅行、学习分享 |
| 核心诉求 | 快速、好看、符合小红书审美风格 |
| 痛点 | 手机端剪辑屏幕小、PC 端专业软件门槛高 |

---

## Alpha 版本核心目标

1. **快速剪辑**：导入视频/图片素材，在时间轴上完成基础剪辑（裁剪、分割、调序）
2. **视觉美化**：应用滤镜、调色、转场，达到小红书审美标准
3. **文字与字幕**：叠加标题文字，并预留自动字幕入口，辅助内容表达
4. **背景音乐**：配乐库快速匹配氛围感音乐
5. **封面制作**：专门的封面图设计（小红书封面是流量入口）
6. **发布准备**：整理标题、正文、话题、封面安全区和版权/水印提醒
7. **导出发布**：导出适合小红书的格式（竖屏 9:16、方形 1:1）；Alpha 阶段为导出流程 Mock

## 版本路线图

```
Alpha (v0.1) — 本阶段目标
  └── 小红书场景核心剪辑功能（Presentation 层 Mock）
  └── 输出导出、字幕、发布准备等流程状态，不做真实后端处理

Beta (v0.2)
  └── 接入基础剪辑后端能力（FFmpeg / 云端渲染）

v1.0
  └── AI 功能接入（自动字幕、AI 配乐推荐、AI 文案）
  └── 小红书一键发布
```

## 竞品参照

| 工具 | 定位 | 与 CopyCut 差异 |
|------|------|----------------|
| 剪映 Web | 通用视频编辑 | 功能过多、未针对小红书优化 |
| Canva | 图文设计为主 | 视频编辑能力弱 |
| 醋溜 / 稿定 | 模板驱动 | 创作自由度低 |
| CopyCut | 小红书剪辑专项 | 轻量、场景化、小红书风格优先 |
