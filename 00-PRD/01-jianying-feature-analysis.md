# 剪映 Web 全功能分析

> 分析基准：剪映 Web 版（2024–2025），以功能域为单位梳理。

---

## 1. 项目管理

| 功能 | 描述 |
|------|------|
| 草稿列表 | 云端保存草稿，跨设备同步 |
| 项目创建 | 新建空白项目 / 从模板创建 |
| 项目设置 | 分辨率、帧率、背景色 |
| 素材管理 | 本地上传、云空间、素材库搜索 |
| 团队协作 | 共享项目、评论、版本历史 |

---

## 2. 时间轴与基础剪辑

| 功能 | 描述 |
|------|------|
| 多轨道时间轴 | 视频主轨、画中画轨、音频轨、文字轨、贴纸轨 |
| 分割 | 将片段在播放头位置一分为二 |
| 裁剪/调整时长 | 拖拽片段两端调整入出点 |
| 删除 | 删除片段，留空或自动吸附 |
| 顺序调整 | 拖拽重排片段顺序 |
| 主轨替换 | 替换素材保留其他设置 |
| 复制/粘贴 | 片段级别复制粘贴 |
| 撤销/重做 | 历史操作栈 |
| 预览播放 | 实时预览，1080p 实时渲染 |
| 缩放时间轴 | 轨道时间轴缩放以精细操作 |

---

## 3. 视频片段属性

| 功能 | 描述 |
|------|------|
| 速度控制 | 正常速度 / 曲线变速（慢镜头、快镜头） |
| 倒放 | 视频反向播放 |
| 旋转 / 翻转 | 90° 旋转、水平/垂直翻转 |
| 画面缩放/位置 | 缩放平移以调整构图 |
| 裁剪（画面比例） | 自由裁剪或固定比例裁剪画面内容 |
| 不透明度 | 调整片段透明度 |
| 混合模式 | 叠加、滤色、正片叠底等 |
| 蒙版 | 矩形、圆形、自由形状蒙版 |
| 关键帧动画 | 位置/缩放/不透明度的关键帧 |
| 定格 | 将某帧定格为静止画面 |

---

## 4. 调色与滤镜

| 功能 | 描述 |
|------|------|
| 基础调色 | 亮度、对比度、饱和度、色温、色调、锐化、高光、阴影、褪色 |
| 曲线 | RGB/单通道曲线调整 |
| HSL | 按色相单独调整饱和度/亮度 |
| 色轮 | 阴影/中间调/高光色轮 |
| 滤镜库 | 数百款预设滤镜，分类浏览（电影、日系、复古、清新等） |
| 滤镜强度 | 滤镜叠加程度滑块 |
| LUT 导入 | 支持导入自定义 LUT 文件 |
| 一键美颜 | 磨皮、美白、瘦脸（适用于人像素材） |

---

## 5. 转场

| 功能 | 描述 |
|------|------|
| 转场库 | 叠化、闪黑、滑动、缩放、故障等上百款 |
| 转场时长 | 可调节转场持续时间 |
| 应用到全部 | 一键统一所有转场 |
| 音效自动匹配 | 部分转场自带音效 |

---

## 6. 文字与字幕

| 功能 | 描述 |
|------|------|
| 添加文字 | 自由文本框，支持多行 |
| 字体库 | 数百款字体，含中文艺术字 |
| 文字样式 | 颜色、描边、阴影、背景框、不透明度 |
| 动画 | 入场/出场/循环动画 |
| 文字模板 | 预设好样式的文字组合 |
| 自动字幕 | AI 语音识别自动生成字幕轨 |
| 字幕编辑 | 逐条修改、合并、分割字幕 |
| 字幕样式批量应用 | 统一修改所有字幕的字体/颜色 |
| 歌词字幕 | 自动匹配背景音乐歌词 |
| 翻译字幕 | 自动翻译为其他语言 |

---

## 7. 贴纸与元素

| 功能 | 描述 |
|------|------|
| 贴纸库 | 表情包、装饰、节日等分类贴纸 |
| 动态贴纸 | 带动画的 GIF 贴纸 |
| 自定义上传 | 导入 PNG/GIF 作为贴纸 |
| 边框 | 视频边框样式 |
| 画中画 | 在主轨上叠加额外视频层 |

---

## 8. 音频

| 功能 | 描述 |
|------|------|
| 背景音乐库 | 按情绪/场景/流派分类，版权授权 |
| 音效库 | 环境音、搞笑音效等 |
| 录音 | 麦克风录制配音轨 |
| 音频提取 | 从视频中提取音频 |
| 音量调节 | 主音量，淡入/淡出 |
| 变声 | 萝莉、男声、机器人等变声效果 |
| 降噪 | AI 自动降噪 |
| 节拍踩点 | 自动检测音乐节拍，辅助卡点剪辑 |
| 变速不变调 | 视频变速时保持原始音调 |
| 音频可视化 | 波形显示 |

---

## 9. 特效

| 功能 | 描述 |
|------|------|
| 视频特效库 | 光效、粒子、故障、电影感等 |
| 画面特效 | 模糊、晃动、局部马赛克 |
| 人像特效 | AI 识别人像后的特效（换脸、抠像等） |
| 背景替换 | AI 抠像后替换背景 |
| 绿幕抠像 | Chroma Key |

---

## 10. AI 功能

| 功能 | 描述 |
|------|------|
| AI 自动字幕 | 语音转文字，多语言 |
| AI 文字转语音 | 多种音色配音，支持情感调节 |
| AI 数字人 | 虚拟主播口播 |
| AI 文案生成 | 根据主题生成脚本/口播文案 |
| AI 一键成片 | 上传素材 + 选模板自动剪辑 |
| AI 智能剪辑 | 自动识别精彩片段 |
| AI 扩图 | 扩展画面边缘 |
| AI 超分辨率 | 视频/图片画质提升 |
| AI 去水印 | 去除画面水印 |
| AI 字幕翻译 | 字幕多语言翻译 |
| AI 抠像 | 人像/物体 AI 精准抠图 |

---

## 11. 模板

| 功能 | 描述 |
|------|------|
| 视频模板库 | 按主题/节日/风格分类的视频模板 |
| 一键替换素材 | 套用模板后替换文字/素材 |
| 创作者上传模板 | UGC 模板生态 |
| 图文模板 | 静态图文海报模板 |

---

## 12. 封面与发布

| 功能 | 描述 |
|------|------|
| 封面截图 | 从视频帧中选取封面 |
| 封面编辑 | 在封面上叠加文字/贴纸 |
| 导出设置 | 分辨率（720p/1080p/4K）、帧率、码率、格式（MP4/MOV/GIF） |
| 水印设置 | 添加/去除剪映水印 |
| 直接发布 | 账号绑定后一键发布到抖音/西瓜 |

---

## 13. 工作区与协作

| 功能 | 描述 |
|------|------|
| 云存储 | 素材和草稿云端同步 |
| 团队空间 | 多人协作编辑 |
| 评论与批注 | 时间轴上的评论标记 |
| 版本历史 | 草稿版本回退 |
| 素材授权 | 音乐/字体商业授权管理 |

## CopyCut 裁剪结论映射

> 本节用于连接剪映全功能分析和 `02-feature-tailoring.md` 的裁剪决策。Step 2 只实现 Presentation Mock，真实处理能力在 Step 3 接入。

| 剪映功能域 | CopyCut 决策 | Step 2 Mock 口径 |
|------------|--------------|------------------|
| 项目管理 | 保留并简化 | 新建项目、比例选择、本地草稿列表 |
| 时间轴与基础剪辑 | 保留核心操作 | 主轨、片段分割/修剪/排序/删除、撤销重做 |
| 视频片段属性 | 简化保留 | 速度、旋转、翻转、缩放/位置放入属性面板，按 P1 处理 |
| 调色与滤镜 | 保留小红书相关能力 | 基础调色 + 小红书风格滤镜，不做专业调色 |
| 转场 | 保留基础转场 | 叠化、滑动、缩放等常用转场 UI |
| 文字与字幕 | 保留文字，字幕先占位 | 文字/模板为核心；自动字幕按钮显示开发中 |
| 贴纸与元素 | 简化保留 | 小红书风格贴纸库，画中画推后 |
| 音频 | 保留背景音乐和基础音量 | 配乐库、试听、音量、淡入淡出；版权状态需提示 |
| 特效 | 大部分砍掉 | 不做复杂视频特效、人像特效、绿幕抠像 |
| AI 功能 | 推后 | Step 2 只保留入口或开发中状态 |
| 模板 | 简化为文字/封面模板 | 不做完整模板生态和 UGC 模板 |
| 封面与发布 | 强化为差异化模块 | 封面制作、发布准备、导出流程 mock |
| 工作区与协作 | Alpha 不做 | 不做团队协作、评论、版本历史、云同步 |
