# 功能裁剪：小红书场景适配

> 基于剪映 Web 全功能集，结合小红书平台内容特征，定义 CopyCut Alpha 版功能边界。

> 口径说明：本文件从产品裁剪角度判断功能是否应进入 CopyCut；具体 Step 2 mock 实施范围以 `05-alpha-scope.md` 为准。产品 P0 不等于 Step 2 真实实现，剪辑、AI、导出等真实处理能力在 Step 3 接入。

---

## 小红书内容特征分析

| 维度 | 特征 | 对产品的影响 |
|------|------|-------------|
| 主要格式 | 竖屏 9:16、方形 1:1 | 默认画布比例，横屏为低优先级 |
| 内容时长 | 15 秒 – 3 分钟 | 时间轴交互不需要为长视频优化 |
| 内容类型 | vlog、穿搭、美食、旅行、分享 | 去除口播/虚拟主播等商业场景 |
| 发布形式 | 视频笔记 + 图文笔记（轮播图） | 需同时支持视频和图片序列两种输出 |
| 视觉风格 | ins 清新、胶片、高饱和、高对比 | 滤镜预设和调色工具需匹配此审美 |
| 封面重要性 | 封面是流量入口，质量高于内容 | 封面制作需作为一级功能单独设计 |
| 文字使用 | 频繁叠加标题、说明文字、心情文案 | 文字模板和快速文字工具是核心 |
| 音乐氛围 | 背景音乐是"情绪载体"，几乎必加 | 配乐库是必备，版权说明清晰 |
| 发布准备 | 标题、正文、话题、封面共同影响点击与分发 | 导出前需要有平台校验和发布素材整理 |

## 功能裁剪矩阵

### ✅ 保留（Alpha 必须）

| 功能域 | 具体功能 | 保留理由 |
|--------|---------|---------|
| **画布格式** | 9:16 竖屏、1:1 方形默认设置 | 小红书主流格式 |
| **基础剪辑** | 分割、裁剪时长、删除、顺序调整、撤销重做 | 剪辑核心工作流 |
| **视频属性** | 速度控制（变快/变慢）、旋转/翻转、画面缩放/位置 | 日常剪辑高频操作 |
| **调色基础** | 亮度/对比度/饱和度/色温/高光/阴影/褪色 | 小红书风格调色必备 |
| **滤镜** | 预设滤镜库（清新/胶片/日系/ins 风分类）、强度调节 | 小红书美学核心 |
| **转场** | 基础转场库（叠化/滑动/缩放）、时长调节 | 短视频必备 |
| **文字** | 添加文字、字体选择、颜色/描边/背景框、入场动画 | 小红书文字频率极高 |
| **文字模板** | 小红书风格预设文字组合（标题/说明/引用款式） | 降低设计门槛 |
| **贴纸** | 小红书风格贴纸库（简洁线条/可爱/打卡类） | 平台调性匹配 |
| **背景音乐** | 配乐库（情绪/场景分类）、音量调节、淡入淡出 | 几乎每条内容必用 |
| **封面制作** | 从视频帧选取封面、封面上叠加文字/贴纸、单独导出封面图 | 小红书封面是流量关键 |
| **图文轮播** | 多张图片组合成轮播序列、每张图独立文字叠加 | 小红书图文笔记形式 |
| **导出** | MP4 导出，1080p，9:16 / 1:1，封面图单独导出（JPG） | 完整发布流程；Step 2 只做 mock |
| **预览** | 实时预览播放、移动端预览框（手机形状预览框） | 所见即所得，减少发布后返工 |
| **发布准备** | 标题/正文/话题草稿、封面安全区、音乐版权和水印提示 | 帮助创作者从“剪完”过渡到“可发布” |

---

### 🔶 保留（Alpha 简化版）

| 功能域 | 具体功能 | 简化说明 |
|--------|---------|---------|
| **自动字幕** | AI 语音识别生成字幕 | Alpha 阶段接口留空，UI 先做 |
| **音效** | 简单音效库（点击/转场/氛围音） | 库内容可精简，仅 20–30 个 |
| **背景替换** | 纯色/渐变/模糊背景（不做 AI 抠像） | 不依赖 AI，用简单方案替代 |
| **倒放** | 视频片段倒放 | 小红书中偶有使用 |
| **不透明度** | 片段和文字层的透明度调节 | 简化，不做混合模式 |
| **HEIC 导入** | iPhone 照片格式支持 | Step 2 可提示转换，真实解码推后 |

---

### ❌ 砍掉（Alpha 不做）

| 功能域 | 砍掉原因 |
|--------|---------|
| 曲线/HSL/色轮（专业调色） | 目标用户无需专业调色工具 |
| LUT 导入 | 小众需求，Alpha 阶段不做 |
| AI 数字人 / 虚拟主播 | 场景不符（小红书用户不做虚拟口播） |
| AI 文字转语音（多音色配音） | Alpha 阶段留空，v1.0 做 |
| AI 超分辨率 / AI 扩图 | 后端复杂，Alpha 不做 |
| AI 去水印 | 合规风险 |
| 关键帧动画 | 专业功能，Alpha 过重 |
| 蒙版 / 混合模式 | 专业功能，Alpha 过重 |
| 绿幕抠像 | 场景不符 |
| 节拍踩点 | v1.0 考虑 |
| 变声 | 场景不符（小红书内容以自然人声为主） |
| 歌词字幕 | 优先级低 |
| 字幕翻译 | Alpha 不做 |
| 团队协作 / 评论 / 版本历史 | Alpha 单人使用 |
| 多平台直接发布 | Alpha 先做导出，发布功能 v1.0 做 |
| 模板 UGC 生态 | 需要社区，Alpha 不做 |
| 商业授权管理 | Alpha 不涉及 |
| 曲线变速（高级） | 简单变速（倍速）即可 |
| 画中画（多层视频） | Alpha 单轨视频为主 |
| 定格帧 | 低频功能，Alpha 不做 |

## CopyCut Alpha 功能架构图

```
CopyCut Alpha
│
├── 项目管理
│   ├── 新建项目（选择比例：9:16 / 1:1 / 16:9）
│   ├── 草稿列表（本地存储）
│   └── 素材导入（本地上传）
│
├── 编辑工作区
│   ├── 预览区（含手机模拟框）
│   ├── 时间轴
│   │   ├── 视频主轨
│   │   ├── 音频轨
│   │   └── 文字/贴纸轨
│   └── 右侧属性面板（随选中元素切换）
│
├── 素材工具栏（左侧）
│   ├── 媒体（视频/图片上传）
│   ├── 音频（配乐库 + 音效）
│   ├── 文字（输入 + 模板）
│   ├── 贴纸（贴纸库）
│   ├── 滤镜
│   └── 转场
│
├── 片段属性（右侧面板）
│   ├── 基础剪辑（速度 / 倒放 / 旋转）
│   ├── 画面（缩放 / 位置 / 不透明度）
│   └── 调色（基础参数）
│
├── 封面制作（独立模块）
│   ├── 从视频帧选取
│   ├── 上传自定义封面图
│   └── 封面文字 / 贴纸叠加
│
├── 发布准备
│   ├── 标题 / 正文草稿
│   ├── 话题标签建议
│   ├── 封面安全区预览
│   └── 音乐版权 / 水印风险提示
│
├── 图文轮播（独立模式）
│   ├── 多图排序
│   ├── 每张图文字叠加
│   └── 统一滤镜应用
│
└── 导出
    ├── 视频导出（MP4，1080p）
    ├── 封面图导出（JPG）
    └── 导出设置（画质 / 帧率）
```

## 阶段实现口径

| 能力类型 | Step 2 Mock | Step 3 Real |
|----------|-------------|-------------|
| 剪辑操作 | 更新时间轴数据、预览状态和属性面板 | 调用真实视频/图片编辑能力 |
| 滤镜/调色 | CSS filter 或静态预览效果 | 输出时真实渲染到视频/图片 |
| 自动字幕 / AI | 入口、按钮状态、开发中提示 | 接入 AI 识别和生成 API |
| 导出 | 参数选择、进度、成功态、示例下载提示 | 真实 MP4/JPG/ZIP 生成 |
| 草稿保存 | 保存轻量项目元数据 | 保存完整工程、素材和云端同步 |
