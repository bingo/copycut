# 用例（Use Cases）

> 聚焦核心工作流的端到端用例描述。

## UC-01：制作一条小红书穿搭视频

**参与者：** 创作者（普通小红书用户）
**前置条件：** 已录好 3–5 段穿搭视频，保存在本地
**后置条件：** 导出 MP4 + 封面 JPG，可直接上传小红书

### 主流程

```
1. 创作者打开 CopyCut，点击"新建项目"
2. 系统弹出比例选择，创作者选择 9:16（竖屏）
3. 系统进入编辑器，创作者点击"导入素材"上传 4 段 MP4
4. 素材出现在媒体面板，创作者依次拖入时间轴
5. 创作者拖动第 2 个片段右端，将其缩短 3 秒（去掉结尾多余内容）
6. 创作者点击片段间隙，添加"滑动"转场
7. 创作者打开"滤镜"面板，选择"日系清新"滤镜，强度 80%，点击"应用到全部"
8. 创作者打开"文字"面板，添加一个文字框，输入"今日穿搭 🌸"，选择字体"思源黑体"，颜色白色，添加描边
9. 创作者打开"音频"面板，选择情绪分类"轻快"，试听后选一首配乐，添加到时间轴
10. 创作者打开"封面"模块，拖动选取器选出最佳帧，在封面上添加文字"换季穿搭合集 / 春日好看显白配色"
11. 创作者打开"发布准备"，填写标题、正文和话题标签，检查封面安全区、音乐版权和水印提示
12. 创作者点击"导出"，选择 1080p，勾选"同时导出封面图"，点击导出
13. Step 2 Mock 中系统显示导出进度和成功态；Step 3 Real 中浏览器弹出下载，创作者获得 video.mp4 和 cover.jpg
```

### 扩展流程

- **4a** 素材格式不支持：系统提示"不支持该格式，请上传 MP4/MOV/JPG/PNG"
- **7a** 创作者不满意滤镜效果：点击"调色"手动微调亮度+10，色温-15
- **11a** 音乐为示例素材：系统提示"请在正式发布前替换为有授权音乐"
- **13a** 导出失败（浏览器限制）：系统提示重试，显示错误原因

## UC-02：制作一组小红书图文轮播笔记

**参与者：** 创作者
**前置条件：** 已拍好 6 张旅行照片，保存在本地
**后置条件：** 导出 6 张带文字叠加的 JPG，可上传小红书

### 主流程

```
1. 创作者新建项目，选择"图文轮播"模式，比例选 1:1
2. 上传 6 张 JPG 图片，在图文编辑界面排序
3. 创作者将第 3 张图片拖到第 1 位（作为封面图）
4. 创作者点击第 1 张图，在图片上添加文字"探店 🗺 东京必去咖啡馆"
5. 依次为其余图片添加说明文字
6. 创作者打开滤镜面板，选择"胶片复古"，点击"应用到全部图片"
7. 创作者在"发布准备"中填写图文笔记标题、正文和话题标签
8. 预览效果满意，点击"导出图片序列"
9. Step 2 Mock 中系统显示导出进度和成功态；Step 3 Real 中系统导出 6 张图片（每张含文字叠加），以 ZIP 打包下载
```

## UC-03：快速套用封面模板

**参与者：** 创作者
**前置条件：** 视频编辑完成，准备制作封面
**后置条件：** 封面图制作完成，可单独下载

### 主流程

```
1. 创作者进入"封面"模块
2. 在帧选取器中浏览视频帧，选择一帧清晰的全身穿搭照
3. 创作者点击"文字模板"标签，浏览小红书封面文字模板
4. 选择"双行标题"模板（上方大标题 + 下方小副标题）
5. 系统自动在封面上应用模板，创作者双击修改文字内容
6. 创作者调整文字颜色使其在图片上清晰可见
7. 点击"导出封面"，下载为 cover.jpg
```

## UC-04：修剪口误片段

**参与者：** 创作者
**前置条件：** 一段录制好的 2 分钟知识分享视频，中间有口误停顿
**后置条件：** 剪掉口误部分，视频更流畅

### 主流程

```
1. 创作者导入视频，拖入时间轴
2. 播放预览，在 0:45 处发现口误停顿
3. 创作者将播放头移至 0:43（口误前），按 B 键分割
4. 将播放头移至 0:48（口误结束后），再次按 B 键分割
5. 选中中间的口误片段，按 Delete 删除
6. 两侧片段自动吸合，继续播放确认连贯
7. 可选：在连接处添加 0.3 秒"叠化"转场遮盖突兀跳接
```

## UC-05：调整视频配乐音量

**参与者：** 创作者
**前置条件：** 视频已有原声人声，同时添加了背景音乐
**后置条件：** 背景音乐不盖过人声，两者平衡

### 主流程

```
1. 创作者选中音频轨道上的配乐片段
2. 右侧属性面板显示音频设置
3. 创作者将音量从 100% 降至 30%
4. 开启"淡入"（1.5 秒）和"淡出"（2 秒）
5. 播放预览确认人声清晰、音乐不喧宾夺主
```

## UC-06：尺寸格式切换

**参与者：** 创作者
**前置条件：** 当前项目为 9:16，希望导出一个 1:1 版本用于小红书方形帖子
**后置条件：** 内容重新适配为 1:1 比例

### 主流程

```
1. 创作者在工具栏点击"画布设置"，选择切换为 1:1
2. 系统弹出提示："画面比例将更改，视频画面会自动缩放以适配，建议手动调整构图"
3. 创作者确认，预览区变为方形
4. 创作者发现人物头部被裁切，选中主轨视频，在预览区向下移动画面
5. 调整后满意，导出 1:1 版本
```

## UC-07：恢复历史草稿

**参与者：** 创作者
**前置条件：** 昨天编辑了一个项目，今天重新打开浏览器
**后置条件：** 上次的编辑状态完整恢复

### 主流程

```
1. 创作者打开 CopyCut 首页
2. 草稿列表显示历史项目（封面缩略图 + 最后编辑时间）
3. 创作者点击目标草稿
4. 系统恢复时间轴、素材位置、调色参数、文字内容等全部状态
5. 创作者继续编辑
```

### 扩展流程

- **4a** 原始素材文件已从本地删除：系统提示"部分素材丢失，请重新导入"，占位符显示在时间轴

## UC-08：批量应用转场

**参与者：** 创作者
**前置条件：** 时间轴上有 8 个视频片段
**后置条件：** 所有片段之间都有统一转场

### 主流程

```
1. 创作者点击任意片段间隙的转场按钮
2. 转场库弹出，选择"缩放"转场
3. 点击"应用到全部片段"按钮
4. 所有 7 个片段间隙都显示转场图标
5. 转场时长统一调整为 0.5 秒
```

## UC-09：导出前完成小红书发布准备

**参与者：** 创作者
**前置条件：** 视频或图文内容已编辑完成
**后置条件：** 发布素材和文案准备完成，导出前风险项清晰

### 主流程

```
1. 创作者点击编辑器顶部或导出弹窗中的"发布准备"
2. 系统展示标题、正文、话题标签编辑区域
3. 创作者输入标题"3套春日通勤穿搭"，正文补充搭配说明
4. 创作者添加话题标签"#穿搭 #通勤穿搭 #小个子穿搭"
5. 系统展示封面安全区预览，提示标题文字略靠近底部
6. 创作者回到封面模块，将标题文字上移
7. 系统展示导出检查项：比例通过、封面通过、音乐版权提醒、水印通过、素材状态通过
8. 创作者确认后进入导出流程
```

### 阶段说明

- **Step 2 Mock**：完成发布准备 UI、检查项状态和提醒文案
- **Step 3 Real**：接入真实素材校验、版权来源、发布 API 或复制到小红书发布页能力
